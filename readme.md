# Running Back-end
```bash
cd backend
npm run start
# or
npm run dev
# or
node .
```

# Front end
## Build frontend
##### React
```bash
npx create-react-app frontend --template typescript

# or

yarn create react-app frontend --template typescript
```

##### NextJS
```bash
mkdir frontend
cd frontend

npx create-next-app@latest --ts
# or
yarn create next-app --typescript
```
## Running front-end
```bash
npm run start # react
# or
npm run dev # nextJS
```
